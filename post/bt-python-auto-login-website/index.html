宝塔面板定时执行Python，自动登录网站，赚取积分

· 2022-01-28 · # 宝塔 # Python

代码参考上篇文章《Windows定时运行Python，实现某网站自动登陆，以赚取积分》
在某个目录新建python脚本和log.txt日志文件，python脚本如下：

# -*- coding: UTF-8 -*-
import json
import requests
import datetime

data = {'username':'你的用户名','password':'你的密码'}
json_data = json.dumps(data) 
result = requests.post(url='https://www.你的登录网址.com:10065/sys/login/pc',data=json_data,headers={'Content-Type':'application/json'})

 #定义运行日志的路径
print_log = open("/www/wwwroot/你的python程序目录/log.txt",'a')

  #输出运行日志
print("请求耗时：{0}, 状态码：{1}, 结果：{2}".format(datetime.datetime.now(),result.status_code,result.text),file = print_log)
print_log.close()

宝塔面板添加计划任务，任务类型“shell脚本”，执行周期“每天 8小时 0分钟”
宝塔面板默认python版本是2.7.5，存在兼容性问题，但执行btpython是3.7.9，所以脚本内容：

btpython /www/wwwroot/替换你的python程序目录/login.py
